# 🌎 Nation Simulator 🛡

This is an **open source ❤** nation simulator with following features:
- Building a nation
- Customizing the nation
- Trading with other players
- Fighting wars
- Forging alliances



## ⚡ Tech Stack

**Client:** React, Redux, CSS

**Server:** Node, Express, MySQL, SQLite, PugJS Template Engine, REST API
